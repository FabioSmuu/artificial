Layer, Neuron : Class
# calculate : Method 
const inputs = [200, 15, 8]

// const example = new Layer(2)
const example = new Neuron()

const output = example.calculate(inputs, ReLU)

console.log(output)

example usage
this example is calculating an array of inputs and returning the result in the ReLU activation function, note in the comments that it is possible to use a Neuron or a Layer.
description
calculate is a method called by the Layer or Neuron class, its use is given by inserting the inputs and the activation function as arguments to return an output.

parameters inputs : Array callback : Function
# setWeights : Method 
const example = new Neuron()

// const example = new Layer(2)
// const weights = [[100, 1, 5], [-200, 5, 10]]

const weights = [100, 1, 5]
example.setWeights(weights)

console.log(example.weights)

example usage
in this example the weights of the neuron created are redefined, note in the comments that it is possible to use a Neuron or a Layer.
description
setWeights is a method called by the Layer or Neuron class, it is used by inserting a numerical array as arguments to define the weights of the neurons.

parameters weights : Array [Number]
